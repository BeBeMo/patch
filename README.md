=====

segmentation fault

##### 此程序有四种类型的段错误：    

程序的跳转依赖于未初始化的变量、内存越界、内存覆盖、内存泄漏。  

##### 共五个错误：  
* 第一个(第30行)：程序的跳转依赖于未初始化的变量。   
原因是数组元素a[3]未赋初值。  

* 第二(第43行)、三个（第44行）：内存越界。  
原因是用malloc分配了10个内存空间，却尝试给第十一个内存空间赋值。  

* 第四个（第53行）：内存覆盖。  
原因是strcpy函数需要一个结束标志‘\0’，无'\0'时需拷贝的大小超出了地址差值，会覆盖之前的值。  

* 第五个（第42行）：内存泄漏。  
原因是用malloc分配的内存空间后，没有用free释放空间。
